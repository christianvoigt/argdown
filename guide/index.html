# Getting Started

If you want to get your feet wet, before jumping right into installing Argdown on your computer, the Argdown sandbox (opens new window) is the right place for you. Otherwise it is recommended to install the VS Code extension.

If you prefer to work with a commandline tool instead, you can jump to installing the Argdown commandline tool.

If you are new to Argdown, we recommend to check out our example reconstruction to get an impression of how it feels to work with Argdown. For a more systematic introduction we recommend to read the syntax documentation.
